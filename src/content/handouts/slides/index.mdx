You can add [revealjs](https://revealjs.com/) slides with the tag: `<Slides>`. Slides have support for [code syntax highlight](https://revealjs.com/code/), [Markdown](https://revealjs.com/markdown/), and [math equations](https://revealjs.com/math/).

<Slides>
  <SlideDeck />
</Slides>

You can fine-tune the slides configurations with the following properties: `hideControls`, `hideControlsTutorial`, `controlsLayout`, `controlsBackArrows`, `progress`, `slideNumber`, and `shuffle`. See more at [revealjs config docs](https://revealjs.com/config/).
